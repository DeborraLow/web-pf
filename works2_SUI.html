SUI
~ for Juice Design Inc. ~
Silhouetto User Interface
User Interface / February 2015
Roles:
・Usability Evaluation
・C# Coding
・Screen Design
Exhibition:
・CEATEC JAPAN 2014
・6-SXSW 2015
The new generation is coming. Juice Design Inc. invented a new way to browse websites with your silhouette hand called "Silhouette User Interface (SUI)". They believe that we will be able to operate a computer without using keyboard and mouse in the future like in SciFi movies, and they also believe they will make it. I evaluated the system of SUI from a useful point of view, and I gave the result of the research as well as the different way of using it. In addition, I suggested and created an entertaiment game using SUI, and it was displayed in 6-SXSW 2015, which an exhibition was held in the US in March 2015.
